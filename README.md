# EuroLeaguePlayerBuilder
The user has to apply the migrations and change the connection string inside the appsettings.json in order to run the app!

EuroLeague Player Builder is a web application that allows basketball enthusiasts to create and manage custom players for the EuroLeague. The app provides an interactive way to build your own roster and manage players across six pre-existing EuroLeague teams.

Key Features:

- Create Custom Players: Users can create new basketball players with custom attributes and assign them to one of the six existing EuroLeague teams.

- Team Information: Each team has a coach, a city, and other relevant details to give context to the roster.

- Player Management: Users can edit or delete only the players they have created, ensuring that default or other usersâ€™ players remain protected.

- Interactive Roster: View team rosters, player details, and manage your created players in a clean and user-friendly interface.

Concept:
The application is designed to simulate building a EuroLeague basketball roster, giving users the freedom to create, customize, and manage players while respecting team structures and ownership rules. It combines fun, strategy, and customization for basketball fans who want to experiment with building their ideal teams.
